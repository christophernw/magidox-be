Spyware and virus check.  Upgrade your computer memory and processor.  If you are using XP, you can try defragment the drive.  Also, you can try right-click My Computer -> Properties -> Advanced -> Settings.  Choose the "Adjust for best performance" option.  Disable the screen saver and wallpaper will also improve the performance.  Disable some start-up programs will help too, but only if you know what you are doing.
